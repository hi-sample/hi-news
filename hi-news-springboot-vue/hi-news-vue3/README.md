# hi-news-vue3

## Version

- node.js  >= 18



## Project Setup

### Install pnpm

```sh
nvm install 20.10.0
nvm use 20.10.0
corepack enable
```

### Install Dependencies

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Compile and Minify for Production

```sh
pnpm build
```
